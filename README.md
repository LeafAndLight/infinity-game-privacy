Privacy Policy – Infinity Game

Last Updated: Nov 21, 2025
Developer / Organization: Leaf and Light Studios
Contact: leafandlightstudio@gmail.com

1. Introduction

Infinity Game is developed by Leaf and Light Studios.
This Privacy Policy explains what data the app collects, how it is used, and how users may request deletion of that data.

2. Summary (Short Version)

Infinity Game does not collect, store, or transmit personal data.

The game works fully offline except for:

In-App Purchases via the Meta Store

The official Meta online leaderboard

No information is sent to Leaf and Light Studios’ servers.

3. Data Collection
3.1 Personal Data

The game does not collect any personal data, including:

Name

Email

Voice

Photos or videos

Location

Biometric data

Any user-generated content

3.2 Meta Account (Oculus Login)

The game only uses Meta’s authentication system to enable:

In-app purchases

Official Meta leaderboard features

Leaf and Light Studios does not store, process, or transmit the player’s Oculus ID.
All authentication and handling are performed solely by Meta.

3.3 Technical Data

Infinity Game does not collect technical or analytics data, including:

FPS

Headset model

Country

Language

IP address

Logs

Analytics

Usage data

Crash reports

Any form of telemetry

4. Save Data

Player progress is stored only:

Locally on the user’s headset, or

Using Meta’s cloud save system, if enabled by the user

No save data is sent to Leaf and Light Studios.

5. Online Connectivity and Leaderboards

The game can be played offline but uses the official Meta leaderboard system.

This means:

Only the score and minimal data required by Meta’s API are transmitted.

All leaderboard data is stored and managed exclusively by Meta.

No leaderboard data is collected or stored by Leaf and Light Studios.

To request leaderboard data removal, users may:

Contact Meta’s official Oculus account support, or

Email leafandlightstudio@gmail.com
 and we will assist with the request using Meta’s available tools.

6. In-App Purchases

All in-app purchases are processed exclusively by Meta.
The game does not receive:

Financial information

Credit card data

Payment history

The game only receives Meta’s confirmation that an item was purchased.

7. Data Sharing

The game does not share data with any third parties.
The only data recipient is Meta, through:

Authentication

In-app purchases

Leaderboard features

8. Data Deletion Requests
8.1 Save Data

Users can delete save data by:

Uninstalling the game

Clearing the app data via the Meta Quest system settings

8.2 Leaderboard Data (Meta)

Users may request deletion of leaderboard data by:

Contacting Meta Support, or

Emailing leafandlightstudio@gmail.com

9. Children’s Privacy

The game does not collect data and is not directed at children under 13.
If future features require data collection, this policy will be updated.

10. Changes to This Policy

Any updates to this Privacy Policy will be published at the same location where this policy is hosted.
